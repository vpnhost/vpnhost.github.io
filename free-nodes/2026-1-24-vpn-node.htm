用户可以通过导入配置文件的方式，快速加载多种代理设置，省去手动输入节点信息的繁琐。无论是从朋友处获取的配置，还是VPN服务商提供的文件，都可以轻松导入，大幅简化使用流程。

2.3 实时流量监控，清晰掌控网络状况

Shadowrocket内置流量监控功能，能够实时显示每个应用的网络使用情况。通过流量分析，用户能够准确判断网络资源分配和流量消耗，便于进行合理优化。

2.4 自定义DNS设置，提升安全与速度

除了代理功能，Shadowrocket支持用户自定义DNS服务器，帮助避免DNS污染，提高访问速度和网络安全，尤其适合面对DNS劫持问题的用户。

4.1 创建新配置文件

打开Shadowrocket后：

  1. 点击左上角的“+”号，进入配置添加页面；

  2. 手动填写代理服务器信息，包括IP地址、端口号、用户认证信息等；

  3. 根据代理协议选择对应的类型（HTTP、SOCKS5等）；

  4. 保存配置，返回主界面。

4.2 代理模式选择

Shadowrocket提供三种主要代理模式，适合不同上网场景：

  • 全局模式：所有网络请求都通过代理服务器，适合访问受限网站；

  • 分流模式：只有特定域名或应用走代理，其他流量直连，提高效率；

  • 直连模式：默认直连，不经过代理，适合无需翻墙的日常浏览。

用户可根据需求灵活切换，达到最佳使用效果。

FAQ

QuantumultX 与 Shadowrocket 相比的最大区别是什么？
QuantumultX 在规则系统和自动化脚本上更灵活，支持丰富的 rewrite 与 filter 功能，而 Shadowrocket 则更轻便易用，适合普通用户快速切换节点。二者可根据使用习惯与功能需求选择。
V2Ray 服务器启动后提示端口被占用，该如何处理？
先用 netstat 或 ss 工具查看占用该端口的进程（例如 `ss -ltnp`），确认是旧进程或其他服务占用。停止占用进程或修改 V2Ray 配置端口，然后重启服务并检查防火墙规则是否开放了新端口。
VLESS + XTLS 模式适合移动网络吗？
XTLS 模式降低加密开销，减少握手次数，适合高延迟或不稳定移动网络。结合策略组和多出口节点，可提升访问速度和连接稳定性，确保用户体验流畅。
Mellow 如何通过配置实现多出口分流？
Mellow 可在配置文件中定义多个 outbound，并结合 routing 规则实现按域名、IP 或策略组分流。这样不同类型的流量可走不同节点，提高使用体验。
Clash Premium 与普通版本有什么区别？
Clash Premium 是增强版，支持更多功能如 TUN 模式、本地 DNS 解析、脚本规则和自定义代理链。它能实现更细致的流量控制和兼容更多协议，适合对网络有高级需求的用户使用。
Nekobox 如何分流多协议流量？
Nekobox 可以同时管理 Shadowsocks、VMess、Trojan 等多协议节点，通过策略组将不同流量分配到指定节点，实现精准分流，提高网络效率和管理便利性。
Hiddify Next 订阅更新后节点消失，如何恢复？
先检查订阅源是否更改格式或被下架，尝试重新粘贴订阅链接并手动刷新；若仍缺失，联系订阅提供方确认是否停用某些节点或使用了新的访问权限机制（例如 token 验证）。
VMess AEAD 加密在安全性方面有何优势？
AEAD 加密可防止数据篡改、重放攻击和流量特征泄露。在高封锁环境下保证节点可用，提升数据安全和连接稳定性，同时降低被检测风险，确保长期安全访问。
WinXray 的多节点分流如何设置？
可通过策略组定义不同节点用途和优先级，结合规则文件将特定域名、IP 或端口流量分流至相应节点，实现游戏、视频或网页加速，并优化整体网络性能和稳定性。
Nekoray 支持 Trojan 吗？
支持。Nekoray 基于 sing-box 核心，能兼容 Trojan、Vmess、Vless、Hysteria2 等多种协议。用户可在同一界面中管理和切换不同类型节点，非常方便。
